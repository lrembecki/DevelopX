# DevelopX

DevelopX challenge project

## Getting Started

### Prerequisities

Application requires docker desktop application

### Installing

Download docker container

```
docker pull lrembecki/developx-19-11-11
```

Run docker container

```
docker run -it --rm -p 80:80 --name developx lrembecki/developx-19-11-11
```

More information are available at [Docker](https://hub.docker.com/r/lrembecki/developx-19-11-11) site
